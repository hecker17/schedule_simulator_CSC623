USAGE:
    python3 main.py <periodicSchedulingAlgo> <aperiodicSchedulingAlgo> <serverPeriod> <serverExecutiontime> <taskFile>

        periodicSchedulingAlgo - 1 is FCFS, 2 is RM, 3 is DM, 4 is EDF, and 5 is LST.
        aperiodicSchedulingAlgo - 1 is Polling Server, 2 is Deferrable Server.
        serverPeriod - Period length for the Server.
        serverExecutiontime - Execution budget available to Server.
        taskFile - Path to task input file.
